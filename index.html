View on GitHub What it does

Simple by design

Built for people who want to rename files without learning a complicated rules engine first.

Quick edits

Ideal for small batches, cleanup tasks, downloaded files, pictures, documents, and messy folders.

Beginner-friendly

The goal is a lightweight experience that feels approachable instead of intimidating.

Why this exists

There are already powerful renaming tools for advanced users. EasyEdits File Renamer is aimed at a different use case: quick, visual, low-friction file renaming when you just need to clean up a folder and move on.
